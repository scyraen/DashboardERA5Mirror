# ERA5 Hourly Weather Data Dashboard

A collaborative university project to visualize global weather patterns using the **ERA5 dataset** via **Google Earth 
Engine (GEE)**, built with **Streamlit** and **Plotly**.

---

## Quick Start
### Clone and Initialize
```bash
# Clone repo
git clone https://git.imp.fu-berlin.de/ct3828fu/dashboardera5.git
cd <Directory>

# Create + Activate venv
python -m venv .venv

# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
